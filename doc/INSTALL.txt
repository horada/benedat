BeneDat
=======
program pro výpočet ceny za poskytnuté služby klientům občanského sdružení


Požadavky:
----------
  * Python (nejlépe 2.6)
  * GTK
  * PyCairo
  * PyGObject
  * PyGTK
  * JagPDF
  * pysqlite

  OS Linux:
    Většina zmíněných programů a knihoven je standardní součástí systému případně je lze jednoduše nainstalovat 
    prostřednictvím balíčkovacího systému dané distribuce (ozkoušeno a vyvíjeno na Ubuntu 9.04 Jaunty Jackalope).
    Myslím že pouze balíček JagPDF je potřeba nainstalovat manuálně. Návod naleznete na stránkách projektu
    (http://www.jagpdf.org/)


  Windows:
    Všechny výše zmíněné knihovny jsou dostupné i pro operační systém Windows (odzkoušeno zkoušeno na Windows XP).
    Instalace však občas činí určité potíže. Mám však odzkoušenou kombinaci:
    pro Python 2.6:
      gtk-2.12.9-win32-2.exe
      python-2.6.2.msi
      pycairo-1.4.12-2.win32-py2.6.exe
      pygobject-2.14.2-2.win32-py2.6.exe
      pygtk-2.12.1-3.win32-py2.6.exe
    případně pro Python 2.5:
      GTK UPŘESNIT
      pycairo-1.4.12-2.win32-py2.5.exe
      pygobject-2.14.2-2.win32-py2.5.exe
      pygtk-2.12.1-3.win32-py2.5.exe
      python-2.5.4.msi


Základní instalace
------------------
Způsobem používaným na dané platformě spustíme soubor benedat.py pomocí interpretu Pythonu.
